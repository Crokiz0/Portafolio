Proyectos

Proyecto 1

Este programa es una calculadora básica que permite al usuario realizar operaciones aritméticas simples como suma, resta, multiplicación y división. El usuario ingresa dos números y selecciona la operación que desea realizar. El programa muestra el resultado de la operación seleccionada.

Tecnologías: Java Core, Manejo de Excepciones.

https://github.com/Crokiz0/Proyectos-Java

Proyecto 2

Este programa calcula la nota que un estudiante necesita en un examen final para alcanzar una nota final deseada. Toma en cuenta las notas de quices, parciales y seguimientos, y realiza cálculos basados en porcentajes predefinidos. Luego, muestra la nota necesaria en el examen final.

Tecnologías: Java Core, Manejo de Excepciones.

https://github.com/Crokiz0/Proyectos-Java

Proyecto 3

Este programa simula un juego en el que el usuario piensa en un número de 4 cifras y el programa "adivina" el número realizando operaciones aritméticas predefinidas. El usuario ingresa su número, el programa realiza las operaciones y luego revierte las operaciones para adivinar el número original.

Tecnologías: Java Core, Manipulación de Dígitos.

https://github.com/Crokiz0/Proyectos-Java
